探索 Vibe Coding

Vibe Coding 的核心概念

💡 意圖驅動

不再一行一行寫代碼，而是描述你想要的「感覺」或結果，讓 AI 來實現。

🚀 快速迭代

接受 AI 生成的代碼，測試結果，不斷調整提示，直到完美。

🌟 無障礙

即使沒有深厚程式背景，也能打造應用程式和原型。

如何開始 Vibe Coding

步驟 1: 選擇工具

Cursor, GitHub Copilot, Claude, Windsurf 等 AI 編輯器。

步驟 2: 描述你的 vibe

例如：「建立一個音樂播放器 App，具有暗黑模式和播放清單功能。」

步驟 3: 迭代與測試

運行、反饋、調整提示。擁抱不完美，享受過程。
